### **0.5.0** (현재 권장 버전)

**이유:**

- 주요 기능이 대부분 완료되었음 (약 70%)
- 테스트 커버리지가 높음 (유닛/통합 100%, E2E 85.7%)
- 성능 최적화가 상당 부분 완료됨
- 문서화가 완료됨
- 하지만 프로덕션 배포를 위해서는 추가 작업 필요

**특징:**

- 베타 버전 (Beta)
- 내부 테스트 및 제한적 배포 가능
- 프로덕션 배포 전 추가 작업 필요
